The source code of successful projects is evolving all the time, resulting in hundreds of thousands of code changes stored in source code repositories. This wealth of data has been used for various mining and learning tasks. However, it is still difficult to search for specific kinds of code changes across a large number of code repositories and changes.

To fill this gap, we here present DiffSearch -- the first scalable and precise search engine for code changes. Given a query that describes possibly abstracted versions of a code snippet before and after a change, the approach returns a set of changes that are guaranteed to match the query.
